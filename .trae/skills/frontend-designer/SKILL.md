# Frontend Designer

This skill provides expertise in frontend design, user interface (UI), and user experience (UX).

## Capabilities

- **Layout Optimization**: Suggests improvements for page structure and layout.
- **Visual Design**: Recommends color schemes, typography, and spacing for better aesthetics.
- **Responsiveness**: Ensures designs work well on various screen sizes.
- **Usability**: Focuses on making interfaces intuitive and easy to use.
- **Code Improvements**: Provides HTML/CSS/JS snippets to implement design changes.

## When to Use

- When the user asks to "redesign" or "improve" the look of a page.
- When the user mentions "UI", "UX", "layout", or "styling".
- When the user wants to make an interface "more practical", "beautiful", or "modern".

## Design Principles

1.  **Simplicity**: Keep the interface clean and uncluttered.
2.  **Consistency**: Use consistent colors, fonts, and styles throughout.
3.  **Feedback**: Provide clear visual feedback for user actions.
4.  **Accessibility**: Ensure the design is accessible to all users.
5.  **Hierarchy**: Use visual hierarchy to guide the user's attention.
